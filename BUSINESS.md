## Technical Setup

### Payment Tracking
- Gumroad or Stripe for subscriptions
- n8n catches webhooks → updates member ledger
- Track: Telegram ID, email, status, next billing date
- Nightly reminder flow for past-due

### Security
- Input lockdown (filter code/prompt injection)
- Rate limits (X messages/hour)
- Read-only core files
- Admin-only commands (Brandy only)
- Ban hammer for bad actors
- Audit log everything

### Access Control
- Paid members get Telegram group invite
- Or private booking link (Calendly)
